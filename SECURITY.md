# Security Policy
If you find a vulnerability or have a security question, please contact contact@wiki.meshtastik-nmsk.ru
Please do not create public issues before discussion.

# Политика безопасности
Если нашли уязвимость или есть вопрос по безопасности, напишите на contact@wiki.meshtastik-nmsk.ru
Просим не создавать публичных issue до обсуждения.
