Эта ветка — краткий обзор того, чем управлять MeshCore‑устройствами: мобильные приложения, веб/ПК и автономные устройства.

## Как выбрать клиент

- **BLE companion + телефон** → обычно самый простой старт.
- **USB‑serial companion + веб/ПК** → удобно для настройки и инфраструктуры.
- **Устройства “всё‑в‑одном”** (например, T‑Deck‑линейка) → могут работать без телефона.
