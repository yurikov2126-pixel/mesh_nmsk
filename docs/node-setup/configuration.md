## Первоначальная настройка ноды

При первом запуске ноды первым делом потребуется выбрать **регион (Region)**. Эта страница — практическая конфигурация Meshtastic для России и типовых сетей. Этот параметр определяет, на каких частотах и с какими ограничениями будет работать радиомодуль. Региональные настройки напрямую влияют на дальность связи, стабильность сети и законность использования оборудования.
:::info
Выбираем регион **Russia**.
:::

Далее нам необходимо дополнительно настроить ноду, настройки могут зависеть от региона и города использования. Ниже приведены настройки для региона Москва/Московская область, 868 MHz.

| Параметр | Значение | Комментарий |
|----|----|----|
| **Region** | RU | Региональная настройка для России |
| **Channels Preset** | LongFast | Используемый местной сетью пресет |
| **POSIX Timezone** | MSK-3 | Московское время |
| **LORA / Modem preset** | LONG_FAST | Используемый местной сетью пресет |
| **LORA / Hop limit** | 7 | Максимальное количество ретрансляций |
| **LORA / TX power** | 20 dBm (SX127x) / 22 dBm (SX126x) | При включённой опции **LAR** SX127x и SX126x это модели чипа LoRa |
| **LORA / Freq slot** | 2 |    |
| **LORA / Freq** | 869.075 МГц | Частота выставляется автоматически при выборе региона RU |

:::warning
### После выставления всех настроек обязательно ознакомьтесь с вкладкой **[Роли](/node-setup/roles)** в меню вики

:::
