## Первоначальная настройка ноды

При первом запуске ноды первым делом потребуется выбрать **регион (Region)**. Эта страница — практическая конфигурация Meshtastic для России и типовых сетей.
Этот параметр определяет, на каких частотах и с какими ограничениями будет работать радиомодуль.
Региональные настройки напрямую влияют на дальность связи, стабильность сети и законность использования оборудования.

:::info
Выбираем регион **Russia**, если мощность узла не превышает 20 dBm, если узел мощнее выберите регион United States

:::

:::info
Обязательно смените Полное и Короткое имя в настройках пользователя

:::

Далее нам необходимо дополнительно настроить ноду, настройки могут зависеть от региона и города использования
Ниже приведены настройки для региона Новомосковск/Новомосковский район, 868mhz

| Параметр                          | Значение                                                                    | Комментарий                                                                                                                  |
|-----------------------------------|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| **LORA / Регион**                 | RU                                                                          | Региональная настройка для России                                                                                            |
| **LORA / Шаблон**                 | LONG_FAST (Большая дальность - быстрый)                                     | Используемый местной сетью пресет                                                                                            |
| **LORA / Колличество прыжков**    | 7                                                                           | Максимальное количество ретрансляций                                                                                         |
| **LORA / Мощность передатчика**   | 20 dBm  / 28 dBm                                                            | Значение устанавливается в зависимости от мощности вашего устройства.<br/>Если устройство мощнее 20 dBm установите регион US |
| **LORA / Частота слота**          | 0                                                                           | Частотный слот необходимо установить вручную                                                                                 |
| **LORA / Переопределить частоту** | 868.888 МГц                                                                 | Частота выставляется вручную                                                                                                 |
| **LORA / OK в MQTT**              | Включить                                                                    | MQTT используется для работы мониторинга и сбора телеметрии, а так же для работы бота                                        |
| **Устройство / Часовой пояс**     | GMT-3                                                                       | Московское время                                                                                                             |
| **Устройство / Роль клиента**     | Client                                                                      | Выберите роль узла, по его назначению. С описанием ролей можно ознакомиться на следующей странице                            |
| **Каналы / LongFast**             | Uplink-Включить<br/>Downlink-Включить<br/>Местоположение-на ваше усмотрение | Настройки общего канала LongFast                                                                                             |


:::info
### После выставления всех настроек обязательно ознакомьтесь с вкладкой Роли в меню

:::
